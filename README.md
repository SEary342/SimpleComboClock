# SimpleComboClock
Simple Combo Clock watchface for Fitbit OS (Versa/Versa Lite)

![screenshot](Simple-Combo-Clock.png)

Features:
-Digital and analog time display
-Multiple color settings
-Toggleable extra date and step information
